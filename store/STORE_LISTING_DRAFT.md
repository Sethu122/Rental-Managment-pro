## App Name

Residential Rental Management System

## Short Description

Offline Windows desktop software for managing residential properties, tenants, rent tracking, arrears, maintenance, and rental reports.

## Full Description

Residential Rental Management System is a professional Windows desktop application for landlords, rental administrators, and small property-management teams that need a clean offline system for day-to-day rental operations.

Manage properties and units, assign tenants, track lease dates and rent amounts, monitor monthly paid and unpaid rent, calculate arrears, log maintenance issues, and export rent summaries to PDF or CSV.

The app runs locally with SQLite, so property data remains on the customer’s computer. White-label branding support lets operators configure the app name, company name, primary color, and logo for their business.

## Key Features

- Property and unit management
- Tenant management with lease dates and rent amounts
- Monthly rent tracking with paid/unpaid status
- Automatic arrears and outstanding balance view
- Maintenance issue tracking by unit
- Dashboard overview for properties, tenants, arrears, and open maintenance
- PDF and CSV rent-summary export
- Offline local SQLite database
- Local administrator setup on first launch
- White-label branding configuration
- Trial and local license support
